Juego de Cartas

Creación del Juego

Este juego de cartas permite que varios jugadores interactuen mediante la apuesta de una conjunto de cartas que serán asignadas a cada uno de ellos de manera aleatoria, dependiendo del poder que tiene cada carta, se obtiene el ganador de la ronda, gana el juego el jugador que obtenga todas las cartas de sus oponentes, cuando un jugador se quede sin cartas, será eliminado del juego.
